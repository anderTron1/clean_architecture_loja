# Projeto de Aprendizado: Clean Architecture

Este é um projeto de estudo para praticar os conceitos de **Clean Architecture** no desenvolvimento de aplicações.  
O objetivo é compreender como organizar o código de forma modular, separando responsabilidades entre domínio, casos de uso e infraestrutura.

## Tecnologias Utilizadas

- Python
- Pytest (para testes)
- Estrutura baseada em **Clean Architecture**:
  - **Domain**: entidades e regras de negócio
  - **Use Cases**: lógica da aplicação
  - **Infra**: persistência e integração com dados externos

## Objetivo do Projeto

- Aprender a estruturar projetos seguindo os princípios da Clean Architecture.
- Implementar boas práticas de separação de responsabilidades.
- Criar testes automatizados para validar o comportamento da aplicação.

## para fazer testes unitarios e de integração
```bash
    python -m pytest --cov=src --cov-report=html
```
